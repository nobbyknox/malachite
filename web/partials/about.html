Change Log

Version 0.9.4 - Released 2015-12-??

New Features and Enhancements
  • Bookmarks can now be starred/unstarred from bookmark tiles.
  • Bookmarks can now be deleted.

Version 0.9.3 - Released 2015-12-13

New Features and Enhancements
  • You can now log out on all browsers and devices with 1 click.
  • You can now request that bookmark thumbnails be refreshed.
  • New background image on the landing page.
  • Firefox add-on for Bookmarkly can now be downloaded from the landing page.
  • Navigation bar is now always visible.
  • Show last 12 recently added bookmarks.
  • Increase bottom margin of page to be more pleasing on the eye.
Bugfixes
  • Bookmarks can now again be updated.

Version 0.9.2 - Released 2015-12-02

New Features
  • Bookmark thumbnails are now created.

Version 0.9.1 - Released 2015-11-30

New Features
  • Bookmarks are now displayed as tiles instead of a dreary table.
Bugfixes
  • You can now reload any page without running into trouble regarding the config that could not be loaded.
  • Set the dateCreated on bookmark creation.

Version 0.9.0 - Released 2015-11-29

New Features
  • Multiple authentication tokens per user. This allows you to log in from various browsers and devices.
  • Notes on bookmarks.
  • Search box now active. Enter three or more characters to activate search.
  • You are now greeted by name.
Bugfixes
  • Remove auth cookie when token is invalid
